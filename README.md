#  Options Trading Dashboard

A modern, interactive options analytics platform built with Dash and Python.  
Designed to replicate a lightweight trading terminal with real-time data, Greek visualizations, and risk analysis tools.

---

## Features

- Black-Scholes pricing engine
- Greek visualizations (Delta, Gamma, Vega)
-  Real-time stock data via yfinance
-  Options chain explorer
-  PnL simulator for strategy analysis
-  Multi-page dashboard with sidebar navigation
-  Sleek dark UI (trading terminal style)
-  Fully dockerized for reproducibility

---

##  Tech Stack

- Dash + Plotly
- NumPy / SciPy
- yfinance
- Docker
